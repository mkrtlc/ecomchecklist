## Visual Style Guide

### Color Palette
| Element | Color | Usage |
|---------|-------|-------|
| Pass/Success | `#22C55E` | Checkmarks, good scores |
| Warning | `#F59E0B` | Warnings, medium issues |
| Fail/Error | `#EF4444` | Critical issues, problems |
| Primary | `#3B82F6` | Brand, CTAs, highlights |
| Background | `#0F172A` | Dark mode base |
| Text | `#F8FAFC` | Primary text |

### Typography
- **Headlines:** Inter Bold / SF Pro Bold
- **Body:** Inter Regular
- **Numbers/Data:** JetBrains Mono (monospace for tech feel)

### Motion Principles
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)` (smooth, not bouncy)
- Duration: 300-500ms for UI elements
- Stagger: 50-100ms between sequential elements
- No excessive bounce or overshoot

## Production Checklist

### Pre-Production
- [ ] Final UI mockups for all screens (Figma)
- [ ] Sample store URL for demo (use staging or permission-granted site)
- [ ] Script voiceover option (if adding later)
- [ ] Music track licensed and downloaded

### Production
- [ ] Screen recordings at 60fps, 4K if possible (downscale later)
- [ ] Clean cursor movements (use mouse smoothing software)
- [ ] Multiple takes of each screen interaction
- [ ] Capture loading states and animations

### Post-Production
- [ ] Motion graphics for Scene 1-3
- [ ] Kinetic typography animations
- [ ] Sound design and music sync
- [ ] Color grading for consistency
- [ ] Export: MP4 H.264, 1080p, 30fps final
- [ ] Export: WebM for web optimization
- [ ] Export: 1:1 square crop for mobile/social
